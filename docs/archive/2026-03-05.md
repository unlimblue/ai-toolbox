## 已知限制

### 当前限制 (等待改进)

1. **Hard Coded 逻辑**
   - `handle_task()` 中固定回复 "领旨，即刻去内阁商议"
   - 无法根据语义动态决定行动
   - 无法灵活选择讨论频道

2. **配置分散**
   - ID 映射在代码中
   - Token 在环境变量
   - 频道配置分散
   - 不支持多组织配置

3. **固定频道限制**
   - Bot 只能在自己配置的频道发言
   - 不能动态选择频道
