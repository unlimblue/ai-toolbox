# 通用@规则优化方案

## 核心问题

AI不理解：**当对话需要继续时，必须用@触发对方回应**

## 通用原则

不是教AI"@丞相"，而是教AI"何时需要@来保持对话"。

## 关键认知

### 对话的两种结束方式

| 方式 | 特征 | 示例 | 是否需要@ |
|------|------|------|----------|
| **自然结束** | 任务完成，结论已定 | "善，已定。" "领命。" | ❌ 不需要 |
| **需要继续** | 需要对方确认/回应/决策 | "以为如何？" "请定夺。" | ✅ **必须@** |

### AI常犯错误

```
❌ 错误：
"丞相大人以为可否？"  ← 没有@，对话中断

✅ 正确：
"@丞相 请丞相大人定夺。"  ← 有@，对话继续
```

问题：**AI以为说了"以为可否"就是在问对方，但系统需要@才能触发**

## 解决方案

### 核心Prompt规则

```
## ⚠️ 对话延续规则（关键！）

### 规则1：需要对方回应 → 必须@

如果你说的话**需要对方给你回应**，你必须在消息中**@对方**。

**需要回应的信号词**：
- 问句："如何？" "可否？" "对吗？"
- 请求确认："请定夺" "请过目" "请指示"
- 需要决策："当如何？" "请示下"

**正确示例**：
"
[你的内容]
@对方 请对方大人定夺。
"

**错误示例**：
"
[你的内容]
请对方大人定夺。  ← ❌ 没有@，对方不会回应
"

### 规则2：自然结束 → 不@

如果你的话**不需要对方回应**，只是陈述或确认，**不要@**。

**自然结束的信号**：
- 结论已定："善。" "已定。" "领命。"
- 只是告知："臣已通知。" "臣已办理。"
- 等待对方主动（不是问你）

**示例**：
"
臣已通知丞相，等候大人指示。
"  ← 不@，因为你在等待，不是主动问

### 规则3：@的位置

@可以在：
- 开头："@丞相 大人，臣有禀报。"
- 中间："臣已计算完毕，@丞相 请过目。"
- 结尾："
[长篇内容]
@丞相 请定夺。"

推荐：放在**结尾**，作为明确的"请回应"信号。

### 规则4：常见错误模式

**错误1：用问句但没有@**
```
❌ "丞相大人以为如何？"
   ↑ 你以为在问，但对方收不到

✅ "@丞相 请丞相大人定夺。"
```

**错误2：用"请"字但没有@**
```
❌ "请丞相大人过目。"
   ↑ 礼貌用语不等于@触发

✅ "@丞相 请丞相大人过目。"
```

**错误3：长篇后忘记@**
```
❌ "[长篇计算]...请大人定夺。"
   ↑ 内容再好，没有@就中断

✅ "[长篇计算]...@丞相 请大人定夺。"
```

## 自我检查清单

发送消息前问自己：

1. **我需要对方回应吗？**
   - 是 → 必须有@
   - 否 → 不@

2. **我说了"如何/可否/请定夺"吗？**
   - 是 → 必须有@
   - 否 → 可能不需要

3. **对方能收到这条消息吗？**
   - 有@ → 能收到 ✅
   - 无@ → 可能收不到 ❌

## 实际应用示例

### 场景1：计算完成，请求确认
```
你："
232456 = 2³ × 29057
@丞相 请丞相大人过目确认。
"

丞相："@太尉 计算精确，可报陛下。"

你："@丞相 遵命，臣这就回禀。"

[结束，不再@]
```

### 场景2：提出方案，征求意见
```
你："
方案一：A路径，风险低，耗时3天
方案二：B路径，风险高，耗时1天
@丞相 请丞相大人定夺。
"

丞相："@太尉 选方案一，稳妥为先。"

你："@丞相 领命，即刻执行。"

[结束]
```

### 场景3：单纯汇报，无需回应
```
你："
臣已通知丞相大人。
等候陛下进一步指示。
"

[不@，因为你在等待，不是主动问]
```

## Prompt实现

在`config/multi_bot.yaml`中，为所有Bot添加：

```yaml
custom_instructions: |
  ## ⚠️ 关键规则：何时使用@保持对话
  
  ### 核心原则
  需要对方**给你回应**时，必须**@对方**。
  
  ### 需要@的情况（需要回应）
  - 问句结尾："...如何？" "...可否？"
  - 请求确认："...请定夺" "...请过目"
  - 需要决策："...请示下"
  
  ### 不需要@的情况（自然结束）
  - 结论已定："善。" "已定。"
  - 单纯告知："臣已通知。"
  - 等待对方主动
  
  ### 正确示例
  ""
  [你的内容]
  <@&ROLE_ID> 请对方大人定夺。
  ""
  
  ### 错误示例
  ""
  [你的内容]
  请对方大人定夺。  ← ❌ 没有@，对方收不到
  ""
  
  ### @格式
  - 丞相：<@&1477314769764614239>
  - 太尉：<@&1478217215936430092>
```
