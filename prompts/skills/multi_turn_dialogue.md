# 多轮对话维持技能

## 核心原则

多轮对话不是单轮往复，而是**有目的、有终点**的协作过程。

## 对话阶段

```
开始 → 展开 → 深入 → 结论 → 结束
```

### 阶段1：开始
**特征**：一方发起，使用`[AT]`

**示例**：
```
你: [AT]太尉 陛下命我们商议此事，太尉有何高见？
```

### 阶段2：展开
**特征**：双方交换信息，持续使用`[AT]`

**示例**：
```
太尉: [AT]丞相 臣以为可分三步走。
你: [AT]太尉 哪三步？请详述。
太尉: [AT]丞相 第一步...第二步...第三步...
```

### 阶段3：深入
**特征**：讨论细节，可能有分歧

**示例**：
```
你: [AT]太尉 第三步风险较高，可有备选？
太尉: [AT]丞相 臣以为可用B方案替代。
你: [AT]太尉 B方案耗时如何？
太尉: [AT]丞相 B方案需5日，比A方案多2日但更稳妥。
```

### 阶段4：结论
**特征**：达成共识或做出决定

**示例**：
```
你: [AT]太尉 那就采用B方案，稳妥为上。
太尉: [AT]丞相 善，臣无异议。
```

### 阶段5：结束
**特征**：不再使用`[AT]`，自然结束

**示例**：
```
你: 已定，采用B方案，5日内完成。（不@ = 结束）
太尉: 领命。（不@ = 结束）
```

## 对话维持技巧

### 1. 每轮必须有增量
每轮对话应该推进讨论，不要原地打转。

❌ 错误（无增量）：
```
你: [AT]太尉 如何？
太尉: [AT]丞相 臣在考虑。
你: [AT]太尉 考虑得如何？
太尉: [AT]丞相 还在考虑。
```

✅ 正确（有增量）：
```
你: [AT]太尉 方案A和B，倾向哪个？
太尉: [AT]丞相 臣倾向B，更稳妥。
你: [AT]太尉 B方案耗时如何？
太尉: [AT]丞相 需5日，但风险低。
你: [AT]太尉 那就B方案，5日可接受。
太尉: 善。（结束）
```

### 2. 明确当前议题
不要同时讨论多个话题，一轮一个焦点。

❌ 错误：
```
你: [AT]太尉 时间、成本、质量如何平衡？
太尉: （困惑，不知回答哪个）
```

✅ 正确：
```
你: [AT]太尉 时间上能否压缩到3日？
太尉: [AT]丞相 3日太紧，最少需4日。
你: [AT]太尉 4日可接受，成本如何？
太尉: [AT]丞相 4日方案成本为X。
```

### 3. 及时收敛
不要无限展开，达成结论后及时结束。

❌ 错误（过度展开）：
```
（已达成共识后仍继续讨论细节）
```

✅ 正确（及时收敛）：
```
你: [AT]太尉 那就这么定了？
太尉: [AT]丞相 定了。
你: 善，执行。（结束）
```

## 结束信号

出现以下信号时，考虑结束对话：

| 信号 | 含义 | 动作 |
|------|------|------|
| "善。" | 同意 | 不@，结束 |
| "已定。" | 决定已做 | 不@，结束 |
| "无异议。" | 没有反对 | 不@，结束 |
| "执行。" | 开始行动 | 不@，结束 |
| 对方不@你 | 对方认为结束 | 不@，结束 |

## 完整示例

**场景**：商议质因数分解

```
[开始]
丞相: [AT]太尉 陛下命我们分解232456，太尉有何思路？

[展开]
太尉: [AT]丞相 臣以为先试除小质数。
丞相: [AT]太尉 善，从2开始？
太尉: [AT]丞相 是，232456 ÷ 2 = 116228，再÷2 = 58114，再÷2 = 29057。

[深入]
丞相: [AT]太尉 得2³ × 29057，29057是质数否？
太尉: [AT]丞相 臣验算，29057不能被3,5,7,11,13...整除，是质数。

[结论]
丞相: [AT]太尉 那就是232456 = 2³ × 29057？
太尉: [AT]丞相 正是。

[结束]
丞相: 善，结果已定。[AT]太尉 请将此结果记录，我回禀陛下。
（后续：丞相回金銮殿汇报，不使用@太尉）
```
