# Skills体系说明

## 什么是Skills

Skills是操作流程，定义**如何做X**：
- **步骤1** → **步骤2** → **步骤3**
- 每个步骤的具体动作
- 检查清单确保不遗漏

## 如何使用Skills

1. SYSTEM_PROMPT会根据场景指引你读取对应的Skills
2. 按Skills描述的步骤执行
3. 每完成一步，心理检查是否到位

## Skills列表

| 文件 | 用途 |
|------|------|
| `at_mention_usage.md` | [AT]标记使用技能 |
| `task_execution.md` | 跨频道任务执行流程 |
| `multi_turn_dialogue.md` | 多轮对话维持技能 |
| `channel_navigation.md` | 频道切换流程 |
| `response_patterns.md` | 回应模板库 |

## 原则

- Skills是**流程**，不是**约束**
- 按Skills执行，但不要机械复述步骤
- 自然、流畅地完成流程
