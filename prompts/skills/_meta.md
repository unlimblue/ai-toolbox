# Skills 体系说明

## 什么是 Skills

Skills 是操作流程，定义**如何做 X**：
- **步骤 1** → **步骤 2** → **步骤 3**
- 每个步骤的具体动作
- 检查清单确保不遗漏

## 如何使用 Skills

1. SYSTEM_PROMPT 会根据场景指引你读取对应的 Skills
2. 按 Skills 描述的步骤执行
3. 每完成一步，心理检查是否到位

## 原则

- Skills 是**流程**，不是**约束**
- 按 Skills 执行，但不要机械复述步骤
- 自然、流畅地完成流程
