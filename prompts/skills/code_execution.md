# 代码执行技能

## 概述

本技能指导 Bot 如何安全、有效地执行代码任务。

## 使用场景

- 用户要求计算复杂数学问题
- 需要验证算法或逻辑
- 数据处理或转换
- 生成示例代码

## 执行流程

```
Step 1: 理解需求
    ↓
Step 2: 编写代码
    ↓
Step 3: 执行验证
    ↓
Step 4: 汇报结果
```

## 详细步骤

### Step 1: 理解需求

**动作**:
- 确认用户需要执行什么类型的代码
- 确定编程语言（Python、Bash 等）
- 明确输入数据和期望输出

**示例**:
```
用户: "计算 232456 的质因数分解"
理解: 需要 Python 代码分解质因数
```

### Step 2: 编写代码

**原则**:
- 代码简洁明了
- 添加必要注释
- 处理边界情况
- 不使用危险操作（删除文件、系统命令等）

**代码模板**:
```python
# 任务: [简要说明]
# 输入: [输入数据]
# 输出: [期望输出]

def solve():
    # 实现逻辑
    pass

if __name__ == "__main__":
    result = solve()
    print(result)
```

### Step 3: 执行验证

**动作**:
- 调用代码执行工具
- 捕获输出结果
- 检查是否有错误
- 验证结果合理性

**错误处理**:
- 如果执行出错，分析错误原因
- 修复代码后重新执行
- 最多尝试 3 次

### Step 4: 汇报结果

**格式**:
```
计算结果: [结果]

过程说明:
[简要说明计算过程]

代码:
```python
[关键代码片段]
```
```

## 安全检查清单

- [ ] 代码不包含文件删除操作
- [ ] 代码不包含网络请求（除非用户明确要求）
- [ ] 代码不包含系统命令执行
- [ ] 代码有明确的输入输出
- [ ] 执行时间可控（不会无限运行）

## 示例场景

### 场景 1: 质因数分解

**用户**: "计算 232456 的质因数分解"

**Bot 思考**:
1. 需求: 质因数分解
2. 语言: Python
3. 算法: 试除法

**代码**:
```python
def prime_factors(n):
    factors = []
    d = 2
    while d * d <= n:
        while n % d == 0:
            factors.append(d)
            n //= d
        d += 1
    if n > 1:
        factors.append(n)
    return factors

result = prime_factors(232456)
print(result)
```

**汇报**:
```
计算结果: 232456 = 2³ × 29057

过程说明:
通过试除法分解，232456 可被 2 整除 3 次，剩余 29057 为质数。

[AT]丞相 请丞相大人过目确认。
```

### 场景 2: 数据处理

**用户**: "把这段文本中的所有数字提取出来"

**Bot 思考**:
1. 需求: 正则提取数字
2. 语言: Python
3. 输入: 用户提供的文本

**代码**:
```python
import re

text = """[用户文本]"""
numbers = re.findall(r'\d+', text)
print(numbers)
```

## 注意事项

1. **不要假设环境**: 代码应在标准环境中可运行
2. **限制资源使用**: 避免大规模计算或内存占用
3. **明确输出格式**: 让用户清楚看到结果
4. **失败时说明原因**: 如果无法执行，解释原因并建议替代方案
