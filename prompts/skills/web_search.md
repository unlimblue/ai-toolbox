# 网页搜索技能

## 概述

本技能指导 Bot 如何使用网页搜索工具获取最新信息。

## 使用场景

- 用户询问需要最新数据的问题
- 需要验证某个事实或数据
- 获取实时信息（天气、股价、新闻等）
- 补充知识盲区

## 搜索流程

```
Step 1: 判断是否需要搜索
    ↓
Step 2: 构造搜索查询
    ↓
Step 3: 执行搜索
    ↓
Step 4: 分析结果
    ↓
Step 5: 整合汇报
```

## 详细步骤

### Step 1: 判断是否需要搜索

**需要搜索的情况**:
- 询问实时数据（天气、股价、汇率）
- 询问最新事件（新闻、更新）
- 超出训练数据时间范围的信息
- 需要验证的具体事实

**不需要搜索的情况**:
- 通用知识（数学公式、历史事实）
- 逻辑推理问题
- 已在上下文中的信息

### Step 2: 构造搜索查询

**原则**:
- 使用简洁的关键词
- 包含必要的时间、地点限定
- 使用中文或英文（根据信息源）

**示例转换**:
| 用户问题 | 搜索查询 |
|---------|---------|
| "今天北京天气怎样？" | "北京 今天 天气" |
| "最新的人民币对美元汇率" | "人民币 美元 汇率 最新" |
| "OpenAI 最近有什么新闻？" | "OpenAI news latest" |

### Step 3: 执行搜索

**动作**:
- 调用搜索工具
- 获取前 5-10 条结果
- 提取标题、摘要、URL

### Step 4: 分析结果

**原则**:
- 优先查看权威来源
- 对比多个来源确认一致性
- 注意信息发布时间
- 识别可能的错误信息

**可信度评估**:
- ⭐⭐⭐⭐⭐ 官方来源（政府、公司官网）
- ⭐⭐⭐⭐ 权威媒体（Reuters、BBC、新华社）
- ⭐⭐⭐ 知名科技媒体
- ⭐⭐ 自媒体、论坛
- ⭐ 未经验证的来源

### Step 5: 整合汇报

**格式**:
```
根据搜索结果:

[结论或答案]

参考来源:
1. [来源名称] - [简要说明]
2. [来源名称] - [简要说明]

[如需要进一步确认，添加 AT 提及]
```

## 示例场景

### 场景 1: 实时数据查询

**用户**: "今天美元对人民币汇率是多少？"

**Bot 思考**:
1. 需要搜索: 是，实时汇率数据
2. 查询: "美元 人民币 汇率 今日"

**汇报**:
```
根据搜索结果:

今日美元对人民币汇率约为 1 USD = 7.24 CNY。

参考来源:
1. 中国外汇交易中心 - 中间价 7.2420
2. 新浪财经 - 现汇买入价 7.2380

[AT]丞相 丞相大人，以上数据供参考。
```

### 场景 2: 事实验证

**用户**: "验证一下: Python 3.12 什么时候发布的？"

**Bot 思考**:
1. 需要搜索: 是，确认具体日期
2. 查询: "Python 3.12 release date"

**汇报**:
```
根据搜索结果:

Python 3.12.0 正式发布于 2023年10月2日。

主要新特性:
- 改进的错误消息
- f-string 语法增强
- 性能提升

参考来源:
1. Python 官方博客 - Release announcement
2. Python Documentation - What's new

数据已确认无误。
```

### 场景 3: 最新资讯

**用户**: "最近有什么 AI 领域的大新闻？"

**Bot 思考**:
1. 需要搜索: 是，最新新闻
2. 查询: "AI artificial intelligence news latest"

**汇报**:
```
根据搜索结果，最近 AI 领域的重要动态:

1. [新闻标题 1]
   [简要摘要]

2. [新闻标题 2]
   [简要摘要]

参考来源:
1. TechCrunch
2. The Verge

[AT]丞相 丞相大人需要臣进一步查询某条详情吗？
```

## 注意事项

1. **明确标注来源**: 让用户知道信息来自搜索
2. **注明时间**: 特别对于实时数据，注明查询时间
3. **处理无结果**: 如果搜索无相关结果，如实告知
4. **避免过度依赖**: 对于常识性问题，不要刻意搜索
5. **隐私注意**: 不要搜索涉及用户隐私的内容

## 常见错误

| 错误 | 正确做法 |
|------|---------|
| 不说明是搜索结果 | 明确说"根据搜索结果" |
| 直接复制大段内容 | 用自己的话总结关键信息 |
| 不验证来源可信度 | 优先引用权威来源 |
| 搜索通用知识 | 常识性问题直接回答 |
